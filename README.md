# Multi-document Embedding Search Engine

**AI Engineer Intern Assignment - CodeAtRandom AI**

A production-ready embedding-based search engine with intelligent caching, vector similarity search, and FastAPI REST interface.

---

## Features

- ✅ **Efficient Embedding Generation** – [sentence-transformers/all-MiniLM-L6-v2](https://www.sbert.net/)
- ✅ **Intelligent Caching** – Pickle-based with SHA-256 hash validation
- ✅ **Vector Search** – NumPy cosine similarity
- ✅ **Result Ranking & Explanation** – Semantic scores + keyword overlap
- ✅ **FastAPI REST API** – Clean, documented endpoints
- ✅ **100-200 Document Support** – 20 Newsgroups dataset
